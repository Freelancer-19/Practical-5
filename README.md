# E-Commerce Backend API

A modular RESTful API built with Node.js and Express, featuring robust data validation and centralized error handling.

## Quick Start Guide

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed (v16 or higher recommended).

### 2. Initialization
Clone this repository or navigate to your project folder, then run:

```bash
# Initialize npm (if you haven't already)
npm init -y

# Install dependencies
npm install express zod

# To run the file
node server.js
